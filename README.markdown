# Rational

Simple calculator for fraction numbers. Four operators are implemented: + - * :

## Usage

`./rational "2/3 + 6/7 - 7/27"`

Result: 239/189

`./rational "2/3 * 1/3 : 4/3"`

Result: 1/6

## Installation

Use roswell for building.
